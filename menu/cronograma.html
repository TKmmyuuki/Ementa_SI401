AULA DATA CONTEÚDO PROGRAMÁTICO
1 07/08 Unidade 0:
Conteúdo: instruções gerais sobre a disciplina.
Objetivos: descrever o funcionamento da disciplina, formato das aulas, Projeto Final, critérios de avaliação e monitorias.

Unidade 1:
Conteúdo: visão geral sobre o funcionamento da web e contextualização da Programação para Web.
Objetivos: descrever os passos de uma requisição entre navegador e servidor, interpretar mensagem HTTP e introduzir o contexto da Programação para a Web.
2 14/08 Unidade 2:
Conteúdo: HTML e estrutura de documentos (tags, DOCTYPE, elementos e atributos).
Objetivos: entender HTML e construir documentos conforme o padrão.

Unidade 3:
Conteúdo: exemplos de elementos HTML (parágrafos, títulos, imagens, hyperlinks, tabelas, listas e iFrames).
Objetivos: construir documentos HTML utilizando elementos comuns.
3 21/08 Unidade 4:
Conteúdo: o elemento HEAD de HTML.
Objetivos: construir documentos HTML com o elemento HEAD.

Unidade 5:
Conteúdo: layouts via HTML.
Objetivos: estruturar o conteúdo em layouts bem definidos.
4 28/08 Unidade 6:
Conteúdo: CSS - definições, sintaxe e formas de utilização.
Objetivos: construir folhas de estilo que definam a aparência de documentos HTML.

Unidade 7:
Conteúdo: CSS - propriedades de fundo de tela, texto e fonte.
Objetivos: explorar propriedades de fundo de tela, texto e fonte com CSS.
5 04/09 Unidade 8:
Conteúdo: CSS - propriedades abreviadas e o modelo de caixas (box model).
Objetivos: definir layouts utilizando o modelo de caixas do CSS.
6 11/09 Unidade 9:
Conteúdo: programação no front-end com JavaScript: definições, sintaxe, instruções e o objeto DOM.
Objetivos: escrever programas básicos em JavaScript para navegadores.

Unidade 10:
Conteúdo: detecção e reação a eventos via JavaScript.
Objetivos: escrever programas que reajam a eventos disparados pelo usuário.
7 18/09 Unidade 11:
Conteúdo: manipulação de formulários HTML via JavaScript.
Objetivos: manipular dados fornecidos pelo usuário via formulários HTML com JavaScript.

Unidade 12:
Conteúdo: orientação a objetos em JavaScript.
Objetivos: seguir o paradigma de orientação a objetos em JavaScript.
8 25/09 Aula para desenvolvimento da segunda parcial do Projeto Final.
- 02/10 Tecnologia em Foco (não haverá aulas).
9 09/10 Unidade 13:
Conteúdo: programação no back-end com PHP: infraestrutura necessária e aspectos gerais da linguagem.
Objetivos: escrever programas simples em PHP executados no servidor.

Unidade 14:
Conteúdo: aspectos específicos da linguagem PHP: operadores, estruturas de controle e funções.
Objetivos: escrever programas em PHP no servidor.
10 16/10 Unidade 15:
Conteúdo: PHP e formulários HTML, Server Side Includes.
Objetivos: escrever programas em PHP que recebam e manipulem dados de formulários HTML.
11 23/10 Unidade 16:
Conteúdo: orientação a objetos em PHP.
Objetivos: seguir o paradigma de orientação a objetos em PHP.

Unidade 17:
Conteúdo: acesso a Bancos de Dados em PHP via PHP Data Objects (PDO).
Objetivos: escrever programas que manipulem bancos de dados com PHP.
12 30/10 Aula para desenvolvimento da terceira parcial do Projeto Final.
13 06/11 Unidade 18:
Conteúdo: Cookies e sessões em PHP.
Objetivos: manter sessões abertas e permitir o fluxo de informações entre páginas.

Unidade 19:
Conteúdo: comunicação assíncrona entre back-end (PHP) e front-end (JavaScript).
Objetivos: trocar dados de maneira assíncrona entre front-end e back-end.
14 13/11 Aula para desenvolvimento da terceira parcial do Projeto Final.
